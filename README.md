### Run Locally

To run **Blogy** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/whoisni30/pinnacle-webdev.git
```

Windows:

```bash
git clone https://github.com/whoisni30/pinnacle-webdev.git
```
